# PPL-JS

A React + TypeScript port of the Prefigure Packing List app (MVP).

## Quick start

1. Clone: `git clone https://github.com/hj-michal-jendrzejewski/PPL-JS`
2. Install: `npm install`
3. Dev: `npm run dev`

This project uses Vite, React, TypeScript, Tailwind CSS, Dexie (IndexedDB) for local persistence and react-dnd for drag & drop.
